Smart Beehive

Smart Beehive helps beekeepers to better care for bees and use data from sensors for getting maximum possible profit per hive.

While development we used the concept of Internet of Things that helped to build the system of hives, which has their own data base and connected to the Internet.


How it Works
The aim of our project
While developing the primary goal was to improve the work of beekeepers and use modern technology in sphere of honey extraction. This will help to understand bees better and overcome the current problem of bee’s disappearance.

Hive

Each hive – a system of sensors that helps to monitor bee’s status.
Humidity and temperature sensors indicate the status of internal microsystem in each hive.
Scales indicate fullness of each hive with honey.
Bee activity sensors includes IR sensors, which can count bees, that flew out or in and also microphone, which registries overall activity of bees in hive.
Description

Hardware level

System of devices, that processing data from sensors (humidity, temperature weight, bee’s activity and battery) and power supply system, which contains of solar panel and battery.

Software level

At software level, we decided to make Android application and Web-interface that allows remotely observe data about each hive.
